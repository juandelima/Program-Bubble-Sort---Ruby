Angka ke - 3 : 1<br/>
Angka ke - 4 : -3<br/>

Hasil tahap-2 : 1 -3 -5 2 3<br/>
Hasil tahap-3 : -3 -5 1 2 3<br/>
Hasil tahap-4 : -5 -3 1 2 3<br/>
